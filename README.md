## 문서 성격에 대한 메모

원본 저장소 README에는 `Claude Opus 4.8`, `Claude Code (Opus 4.8)`, `Claude Code & Cowork`가 별도 항목으로 나뉘어 있습니다. 또한 `Claude.ai` 항목에는 `Anthropic Reminders`가 따로 연결되어 있습니다.

따라서 처음 번역했던 `Anthropic/claude-opus-4.8.md`는 Claude Code 전용 시스템 프롬프트라기보다는 웹/모바일의 `claude.ai` 또는 Claude 앱 채팅 제품 쪽에서 쓰이는 시스템 프롬프트로 보는 것이 더 타당합니다. 본문에도 웹/모바일 채팅 인터페이스, Claude 앱, Artifacts, 메모리, 과거 채팅 검색, claude.ai 관련 제약과 같은 소비자용 채팅 제품 기능이 다수 등장합니다.

반대로 `Anthropic/claude-code-opus-4.8.md`와 `Anthropic/claude-code.md`는 원본 저장소 README에서 Claude Code 항목으로 분리되어 있으므로 Claude Code 관련 시스템 프롬프트로 볼 수 있습니다. `Anthropic/claude-cowork.md`와 `Anthropic/claude-cowork-dispatch.md`는 Cowork 및 Cowork Dispatch 관련 문서로 분리되어 있습니다.
