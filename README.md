## 문서 성격에 대한 메모

제공된 원문 링크와 저장소 README에서는 이 파일을 `Claude Opus 4.8`의 `System prompt`로 표기하고 있으며, 해당 파일 자체가 `Claude Code` 전용 시스템 프롬프트인지 `claude.ai` 채팅 제품용 시스템 프롬프트인지 명시적으로 구분하지는 않습니다.

다만 본문에는 웹/모바일 채팅 인터페이스, Claude 앱, Artifacts, 메모리, 과거 채팅 검색, claude.ai 관련 제약과 같은 소비자용 채팅 제품 기능이 다수 등장합니다. 반면 Claude Code는 Claude에 접근할 수 있는 여러 제품 중 하나로 언급되는 정도입니다.

따라서 이 문서는 Claude Code 전용 시스템 프롬프트라기보다는 `claude.ai` 또는 Claude 앱의 채팅 제품 쪽에서 쓰이는 시스템 프롬프트로 추정됩니다.
